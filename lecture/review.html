2. 基本構造（おまじない）

要素意味と役割
#includeライブラリの読み込み命令
stdio.h標準入出力（printf等）を可能にするヘッダファイル
int main(void)プログラムの実行開始地点（エントリーポイント）
#include <stdio.h> // 必須ツールセット

int main(void) { // 処理開始
    ... 処理 ...
    return 0; // 終了報告
}

3. データの型と出力

型用途書式指定子
int整数%d
double実数（小数）%f
char一文字%c
⚠️ 基本ルール
  • 文末は必ずセミコロン ;
  • 全角スペースはエラーの元
  • \n で改行を実行
    • ※ \（バックスラッシュ）は環境により ¥（円記号）と表示されますが同じ意味です。

4. 条件分岐 (if)

演算子意味
a == b等しい
a != b等しくない
a > baが大きい
a >= ba以上
if (score >= 60) {
    printf("合格\n");
} else {
    printf("不合格\n");
}

5. 繰り返し (for / while)

for文：回数指定

for (int i = 0; i < 5; i++) {
    printf("%d 回目\n", i + 1);
}

while文：条件継続

while (条件) {
    // 真の間繰り返す
}
🔍 実践：組み合わせ

for の中で if を使うことで、「特定の条件（偶数など）の時だけ処理する」といった高度な制御が可能になります。

6. 総合サンプル（1〜5の組み合わせ）

1から10まで数え上げ、それぞれが偶数か奇数かを判定して表示するプログラムです。

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            printf("%d は偶数です\n", i);
        } else {
            printf("%d は奇数です\n", i);
        }
    }
    return 0;
}
💡 学習のポイント
  • i % 2 == 0 : 2で割った余りが0 ➔ 偶数の判定
  • for ➔ if/else の入れ子構造（ネスト）を理解しましょう。
