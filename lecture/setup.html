GETTING STARTED

Environment Setup

C言語プログラミングを始めるための準備

1. ソフトウェアのインストール

コードを書くための「エディタ」と、実行するための「コンパイラ」を導入します。

VSCode (エディタ)

世界中で使われている高機能エディタです。ここにプログラムを記述します。

Download VSCode

Code::Blocks (コンパイラ同梱版)

C言語のコンパイラ（GCC）を簡単に導入するために使用します。

Download Code::Blocks
⚠️ 重要：ファイル選びの注意点

Code::Blocksをダウンロードする際は、必ずファイル名に 「mingw」 が含まれているものを選んでください。
例：codeblocks-25.03mingw-setup.exe
※ mingwが入っていない通常版では、コンパイラがインストールされないため実行できません。

2. パスの設定（環境変数）

VSCodeからGCCコンパイラを使えるように設定します。自分がインストールした方法に合わせて、追加する「パス」を選んでください。

A. MSYS2 (pacman)

MSYS2を導入してコマンドでgccを入れた人

C:\msys64\mingw64\bin
※ または C:\msys64\ucrt64\bin

B. Code::Blocks同梱版

「mingwが入っているもの」をインストールした人

C:\Program Files\CodeBlocks\MinGW\bin
🛡️ インストールで困った場合

ウイルス対策ソフトの影響で MSYS2 のインストールや実行がブロックされてしまう場合は、パターンB（Code::Blocks同梱版） を試してください。こちらの方が環境によって安定して動作する場合があります。

  • 1
    環境変数の設定を開く
    Windowsキーを押し、検索窓に「環境変数」と入力。「システム環境変数の編集」をクリックします。
  • 2
    Pathを編集
    「環境変数」ボタンを押し、システム環境変数の「Path」を選択して「編集」をクリックします。
  • 3
    新規追加
    「新規」を押し、上記 A または B のパスを貼り付けて「OK」で閉じます。
💡 設定後の注意

環境変数を設定した後は、VSCodeを一度完全に閉じて再起動してください。再起動しないと設定が反映されません。

3. エクスプローラの設定

ファイルの末尾（.c や .exe）が見えないと開発に支障が出るため、拡張子を表示させます。

💡 設定手順
  • エクスプローラ（フォルダ）を開く
  • 上部の「表示」タブ（または「…」メニュー）をクリック
  • 「表示」➔「ファイル名拡張子」にチェックを入れる

4. VSCodeの日本語化と拡張機能

より使いやすくするために、日本語化とC言語用の補助機能を導入します。

日本語化の設定

  • 1
    左側のアイコンから「Extensions (拡張機能)」を選択（または Ctrl+Shift+X）
  • 2
    検索窓に "Japanese" と入力
  • 3
    "Japanese Language Pack" をインストールし、右下のポップアップで "Change Language and Restart" をクリック

C言語の拡張機能

コードの補完や実行を補助する必須ツールです。

1. C/C++ (Microsoft製)

コードの補完やエラー表示をしてくれます。"C/C++" で検索してインストールしてください。

2. Code Runner

右上の再生ボタンで簡単に実行できるようになります。
※ 重要：インストール後、歯車アイコン ➔ 「拡張機能の設定」から "Run In Terminal" にチェックを入れてください。

5. VSCodeの基本操作

プログラミングを開始する際の基本的な流れです。

1. フォルダを開く

「ファイル ➔ フォルダを開く」で、自分の作業用フォルダを必ず指定します。1ファイルだけで作業せず、フォルダ単位で管理するのが基本です。

2. ファイル作成と保存

新しいファイルを作成し、必ず main.c のように .c を付けて保存します。保存は Ctrl + S です。

3. ターミナルで実行

VSCode下部のターミナル（表示されない場合は Ctrl + Shift + @）で以下のコマンドを打ちます。

# コンパイル（ビルド）
gcc main.c -o app.exe

# 実行
./app.exe

6. 動作確認

すべてが正しく設定されたか、以下のコードをコピーして実行してみましょう。

📄 verification.c
#include <stdio.h>

int main(void) {
    printf("Hello, C Programming!\n");
    return 0;
}

実行手順の再確認

  1. VSCodeで新しいファイルを作り、verification.c という名前で保存する。
  2. ターミナルを開き、gcc verification.c -o test.exe と打つ。
  3. エラーが出なければ、./test.exe と打って実行する。
  4. 画面に Hello, C Programming! と表示されれば成功です！
