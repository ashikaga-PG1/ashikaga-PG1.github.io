# Quiz 9
## Q1
関数を main より後ろに書くとき、先に必要になるものは？

- A: #define
- B: 関数プロトタイプ宣言
- C: malloc
- D: goto

**Answer:** B
**Explain:** 先に「この関数があります」という宣言（プロトタイプ）が必要です。

## Q2
Cの引数渡しとして基本なのは？

- A: 参照渡し
- B: 値渡し
- C: ポインタ渡しだけ
- D: コピー禁止

**Answer:** B
**Explain:** Cは基本は値渡し（コピー）です。

## Q3
戻り値がない関数の戻り値型は？

- A: int
- B: double
- C: void
- D: null

**Answer:** C
**Explain:** void は戻り値がないことを表します。

## Q4
関数内で宣言した変数（ローカル変数）の有効範囲は？

- A: プログラム全体
- B: その関数の中だけ
- C: 同じファイル全体
- D: OS全体

**Answer:** B
**Explain:** ローカル変数はブロック（関数）を出ると使えません。

## Q5
関数を使う最大の利点として最も適切なのは？

- A: 実行が必ず速くなる
- B: 同じ処理を再利用し、見通しが良くなる
- C: 必ず短いコードになる
- D: printfが不要になる

**Answer:** B
**Explain:** 再利用と分割で、保守性が上がります。
