# Quiz 6
## Q1
Cの配列 a の最初の要素の添字は？

- A: 1
- B: 0
- C: -1
- D: aの長さ

**Answer:** B
**Explain:** 配列の添字は0から始まります。

## Q2
要素数5の配列 a[5] の最後の添字は？

- A: 5
- B: 4
- C: 0
- D: 3

**Answer:** B
**Explain:** a[0]〜a[4] の5個です。

## Q3
配列の全要素を順に処理する for ループとして適切なのは？

- A: for(i=1;i<=5;i++)
- B: for(i=0;i<5;i++)
- C: for(i=0;i<=5;i++)
- D: for(i=5;i<0;i--)

**Answer:** B
**Explain:** 範囲外アクセスを避けるには i<要素数 が基本です。

## Q4
範囲外アクセスが起きる例として正しいのは？

- A: a[0]に代入
- B: a[4]を参照
- C: a[5]を参照
- D: a[i]を参照

**Answer:** C
**Explain:** a[5] は5要素配列では範囲外です。

## Q5
配列の平均を求めるとき、合計 `sum` を求めたあとに行う計算として適切なのは？

- A: sum + N
- B: sum - N
- C: sum / N
- D: sum * N

**Answer:** C
**Explain:** 平均は「合計 ÷ 要素数」で求めるので、`sum / N` です。
