## Q2
`R <= 0` のデータが出てきたときの処理として適切なのは？

- A: そのまま I=V/R を計算する
- B: `"R error"` を表示してスキップする
- C: 必ず 0 を代入する
- D: ファイルに保存する

**Answer:** B
**Explain:** 課題3では不正な抵抗値をエラー表示して飛ばします。

## Q3
`double f(double V,double R)` の引数は何？

- A: 電圧と抵抗
- B: 電流と電圧
- C: 抵抗と抵抗
- D: ファイル名

**Answer:** A
**Explain:** `calcI(V, R)` のように電圧と抵抗を受け取ります。

## Q4
有効データが 0 件のとき、平均の表示はどうする？

- A: avg=0
- B: avg=-1
- C: avg=NA
- D: 表示しない

**Answer:** C
**Explain:** 仕様では有効データが 0 件なら NA を表示します。

## Q5
`V_list={5.0}`, `R_list={0.0}` の1件だけを処理したとき、最後の平均表示はどうなる？

- A: avg=0.000
- B: avg=R error
- C: avg=NA
- D: avg=5.000

**Answer:** C
**Explain:** `R <= 0` は無効データとしてスキップし、有効件数が 0 なので `avg=NA` になります。
