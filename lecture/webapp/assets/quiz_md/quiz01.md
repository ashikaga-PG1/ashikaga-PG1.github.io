# Quiz 1
## Q1
C言語プログラムの実行が開始される関数はどれ？

- A: start()
- B: main()
- C: printf()
- D: init()

**Answer:** B
**Explain:** C言語では main 関数から処理が始まります。

## Q2
printf を使うために必要なヘッダはどれ？

- A: <math.h>
- B: <stdio.h>
- C: <stdlib.h>
- D: <string.h>

**Answer:** B
**Explain:** printf は標準入出力の機能なので <stdio.h> をインクルードします。

## Q3
文字列 "Hello" の後に改行して表示したい。適切なのは？

- A: printf("Hello");
- B: printf("Hello\n");
- C: printf("\nHello");
- D: printf("Hello\t");

**Answer:** B
**Explain:** \n は改行コードです。

## Q4
IPO の意味として正しい組み合わせは？

- A: Input→Print→Output
- B: Input→Process→Output
- C: Init→Process→Out
- D: Input→Program→Order

**Answer:** B
**Explain:** プログラムは入力→処理→出力の流れで考えると整理しやすいです。

## Q5
main の最後に return 0; と書く意味として最も近いのは？

- A: メモリを全消去する
- B: 正常終了をOSに伝える
- C: 必ず0を表示する
- D: 変数を初期化する

**Answer:** B
**Explain:** return 0 は「正常終了」の慣習的な戻り値です。
