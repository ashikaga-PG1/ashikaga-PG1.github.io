# Quiz 3
## Q1
条件分岐で「等しい」を判定する演算子は？

- A: =
- B: ==
- C: !=
- D: <=

**Answer:** B
**Explain:** = は代入、== が等しい比較です。

## Q2
n が偶数かどうかを判定する条件として適切なのは？

- A: n%2=0
- B: n%2==0
- C: n/2==0
- D: n%2!=2

**Answer:** B
**Explain:** % は余り。余りが0なら偶数です。

## Q3
for 文の基本形に含まれる3要素は？

- A: 宣言・実行・終了
- B: 初期化・条件・更新
- C: 入力・処理・出力
- D: 比較・代入・表示

**Answer:** B
**Explain:** for(初期化; 条件; 更新) が基本形です。

## Q4
while 文の説明として正しいのは？

- A: 必ず1回は実行される
- B: 条件が真の間、繰り返す
- C: 回数が決まった繰り返し専用
- D: 条件が偽の間だけ実行

**Answer:** B
**Explain:** while は条件が真の間、繰り返します。

## Q5
無限ループが起きやすい原因として最も多いのは？

- A: printfを書いた
- B: ループ変数の更新を忘れた
- C: 変数をintにした
- D: returnを書いた

**Answer:** B
**Explain:** 条件が変化しないと、いつまでもループが終わりません。
