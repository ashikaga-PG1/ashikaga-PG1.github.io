# プログラミング１ 統合教育ポータル（2026年度版）

このディレクトリは、C言語学習のための「ポータル画面」「授業ツール」「デバッグゲーム」を統合したプラットフォームのソースコードです。GitHub Pages (`ashikaga-PG1.github.io`) での公開を前提に、静的ファイルのみで構成されています。

## 📁 フォルダ構成

- `index.html` / `styles.css` : **メインポータル**。全ツールへの入り口です。
- `lecture/` : **授業資料ツール (Lecture Tool)**
  - `index.html` : ツールの本体。
  - `webapp/` : スクリプト、スタイル、講義データ（Markdown/画像）。
  - `secret/` : **制限エリア (Answer Area)**。課題の解説・解答を掲載。
- `game/` : **Circuit Quest**。電気回路の不具合を修正するデバッグゲーム。

## 🔑 アクセス制御（解説・解答編）

課題の解説編は、学生の学習進度に合わせて個別にロックを解除できるように「別建て」の制限エリアに集約しています。

## 🛠 メンテナンス・更新方法

### 1. 講義タイトルの変更
`lecture/webapp/data/bundle.js` 内の `sessions` 配列を編集することで、授業ツール内のタイトルや構成を更新できます。

### 2. 解説コンテンツの更新
`lecture/secret/index.html` 内の `loadContent(n)` 関数内の HTML 文字列、または外部ファイル化して読み込むように変更することで、各課題の解答詳細を更新できます。

### 3. デザインの統一
全てのページで「ミッドナイト・ブルー」と「ガラスモーフィズム」のテーマを使用しています。基本カラーは以下の通りです：
- 背景: `#030a16`
- アクセント（青）: `#3b82f6`
- アクセント（金）: `#f59e0b`（制限エリア用）
